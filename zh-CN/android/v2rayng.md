# 📱 v2rayNG for Android

> 🚀 **Android 轻量客户端** | 导入订阅快、连接直观、兼容性高

[v2rayNG](https://github.com/2dust/v2rayNG) 是 Android 平台常用的代理客户端，适合希望快速完成订阅导入并直接连接节点的用户。

## ✨ 功能特点

### 🔧 支持协议

| 协议 | 状态 | 说明 |
|------|------|------|
| VMess | ✅ | 常见 V2Ray 协议 |
| VLESS | ✅ | 新版主流协议 |
| Trojan | ✅ | 伪装流量方案 |
| Shadowsocks | ✅ | 兼容旧节点 |
| SOCKS | ✅ | 通用代理支持 |

### 📋 系统要求

- 操作系统：Android 8.0+
- 权限需求：网络权限、VPN 权限
- 存储空间：100MB 可用空间

## ❓ 常见问题

**Q: 导入报错或订阅为空？**  
A: 链接可能失效或不完整；先浏览器测试链接，再重新导入。

**Q: 连上后打不开网页？**  
A: 切换节点重试，检查 VPN 权限是否仍在，确认本地时间准确。

**Q: 节点全部超时？**  
A: 先切换网络，再更新订阅；如仍异常，联系节点提供方排查。
